🚀 Quick Start Guide

  1. Using Startup Launcher: Double-click the run.bat script. It automatically verifies Python, sets up a virtual environment (.venv), installs all required dependencies from requirements.txt, and launches the application.
  2. Manual Terminal Setup: 
    # 1. Create virtual environment
python -m venv .venv

# 2. Activate virtual environment
# On Windows:
.venv\Scripts\activate.bat
# On macOS/Linux:
source .venv/bin/activate

# 3. Install packages
pip install -r requirements.txt

# 4. Start FastAPI application
uvicorn app.main:app --reload
  3. Testing & Using: Write or paste your source code (Python, JavaScript, or Java) into the editor on the screen. The AI analysis will instantly run, highlighting bugs, style errors, security flaws, time/space complexities, and providing clean code suggestions!

Understanding Time & Space Complexity

O(1) - Constant Time / Space

Execution time or auxiliary storage space remains constant regardless of the input size.

def get_first(items):
    return items[0] # Always 1 step
O(N) - Linear Time / Space

Execution time or memory grows linearly with the size of the input dataset.

def linear_search(items, target):
    for item in items: # N iterations
        if item == target:
            return True
O(N^2) - Quadratic Time

Execution time is proportional to the square of the input size. Typically occurs with nested loops.

def print_pairs(items):
    for x in items:
        for y in items: # Nested loop
            print(x, y)
O(2^N) - Exponential Time

Growth doubles with each addition to the input dataset. Typically seen in branching recursive calculations.

def fibonacci(n):
    if n <= 1: return n
    return fibonacci(n-1) + fibonacci(n-2) # Branches!
